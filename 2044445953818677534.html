使用 yt-dlp 下载视频

.\yt-dlp --js-runtimes node --cookies cookies.txt --write-thumbnail --embed-thumbnail "video_url"
参数作用说明
.\yt-dlp执行当前目录下的 yt-dlp 程序。
--js-runtimes node指定 JavaScript 运行环境为 Node.js。某些网站（如 YouTube）使用加密的 JS 脚本来限制下载，yt-dlp 需要调用外部 JS 环境来解析这些算法。
--cookies cookies.txt读取同目录下的 cookies.txt 文件。这通常用于下载需要登录才能查看的视频（如会员内容、高画质限制内容或私人视频）。
--write-thumbnail下载视频的同时，额外保存一份封面图片文件到本地。
--embed-thumbnail将下载的封面图直接写入视频文件的元数据中。这样你打开视频时，文件图标就会显示为视频封面。
"video_url"占位符，执行时需要替换成真实的视频网址。

专栏：无
